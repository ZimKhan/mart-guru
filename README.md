# QuwanShop
一款适合微商及代理商销售产品的在线商城，可自定代理等级、折扣，每个代理拥有专属商城页面用于分享推广赚钱。使用react-native 0.57.5版本。长期维护版本，欢迎大家踊跃提交贡献代码；

## 安装
```bash
npm install

react-native run-ios

react-native run-android
```

APP端开源，服务端收费部署与APP对接。
